Create Calendar Table, and Refresh Date Table

Adding calendar date table


In order to create a calander Date table we use the following DAX
Calendars = ADDCOLUMNS ( CALENDARAUTO (),
 
    "year",YEAR([Date]),
 
    "MonthNo", MONTH([Date]),
 
    "Month", FORMAT([Date],"mmm"),
 
    "Quarter", FORMAT([Date],"\QQ"),
 
    "YearMonth", FORMAT([Date],"YYYY-MM"),
 
    "YearMonthNAME", FORMAT([Date],"YYYY/mmm"),
 
    "WeekdayNo", WEEKDAY([Date],2),
 
    "Weekday", FORMAT([Date],"ddd"))

Adding Refreshing date table

1.       In Power BI Desktop Home ribbon, click on Get data and select Blank Query. Power Query will open.
2.       Under Properties, replace Query 1 by Refresh Date in the Name box.
3.       Next, go to the formula bar and type the following DAX code and click enter: =Date Time. Local Now ()
4.       The current date and time will be displayed. Click on Convert to Table under the Date Time Tools Tab. 
5.      After the refresh table pops up, Rename Column1 as Refresh Date 
6.      Click on Close & Apply to save.
